# Product Overview

## Photo Search Application - "Living Museum Interface"

A commercial prototype for a pro-grade AI photo workstation that provides intelligent photo search and organization capabilities.

### Core Value Proposition
- **Local-first AI processing** - No data mining, privacy-focused
- **Semantic search** - Find photos using natural language descriptions
- **Comprehensive metadata extraction** - Extract all possible metadata from images and videos
- **Real-time indexing** - Automatic detection and processing of new files
- **Professional workflow** - Built for photographers and media professionals

### Key Features
- **Multi-modal search**: Text-to-image, image-to-image, and metadata-based search
- **Real-time file watching** - Automatically indexes new files as they're added
- **Comprehensive format support** - Images (JPG, PNG, HEIC, RAW, etc.) and videos (MP4, MOV, etc.)
- **3D visualization** - Interactive photo browsing with Three.js
- **Export capabilities** - Batch export of selected photos
- **Pricing tiers** - Commercial model with usage-based pricing

### Architecture Philosophy
- **Modular design** - One task = one file principle
- **Backend-focused** - Robust Python API with React frontend
- **Documentation-driven** - Comprehensive docs for every component
- **Commercial focus** - "Studio" feature set as premium differentiator

### Target Users
- Professional photographers
- Media organizations
- Content creators
- Anyone with large photo collections requiring intelligent organization
