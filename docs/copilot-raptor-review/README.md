# Copilot Raptor Review — Consolidated Docs

This folder contains the review documents produced by GitHub Copilot (Raptor mini Preview) focusing on UX, toggles guidance, privacy & consent, Tauri Mac packaging guidance, and developer handoff checklists.

Files:

- `TOGGLE_EXPLANATION.md` — Detailed suggestions for search toggle UX (semantic vs metadata vs hybrid).
- `PRIVACY_AND_CONSENT.md` — Local-first, cloud opt-in, provider consent, and telemetry guidance.
- `TAURI_MAC_GUIDANCE.md` — Packaging and Mac-first Tauri guidance with MPS & file permission notes.
- `DESIGN_AND_COPY.md` — Suggested UI copy for toggles, tooltips, error messages, and quick help tips.
- `DEV_HANDOFF_CHECKLIST.md` — Developer tasks, testing checklist, and QA scenarios.
- `API_AND_UI_SUGGESTIONS.md` — Suggested API enhancements for toggles & job status endpoints.

Use these as ready-to-use artifacts for handoff to designers and engineers.
